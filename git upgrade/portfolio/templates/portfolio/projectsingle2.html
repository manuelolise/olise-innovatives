A Machine Learning Project

Weather Prediction Model using Logistic Regression, SVM,.......

Machine-learning algorithms use statistics to find patterns in massive amounts of data. And data, here, encompasses a lot of thingsâ€”numbers, words, images, clicks, what have you. If it can be digitally stored, it can be fed into a machine-learning algorithm. Machine learning comes in three different types which is supervised learning, unsupervised learning and reinforcement learning.

In supervised learning, the most prevalent, the data is labeled to tell the machine exactly what patterns it should look for. An example is the Netflix movie platform, which uses machine learning in other to find out similar movies that you would like to see based on your previous movies watched. We also have unsupervised learning and reinforcement learning, but for the purpose of this project i will focus on supervised learning.

This project is a supervised machine learning project of predicting whether rain will fall on a specific day based on various features and data that has been read, arranged, trained and tested by the machine learning algorithms. For this project i made use of four machine learning algoirthms (Support Vector Machine(SVM), Logistic Regression(LR), Decision Tree Classifier(DTC), Random Forest Classifier(RFC)) and calculated their accuracy score based on the most effective in predicting the outcome. The datasets used for this project was gotten from kaggle website and mined from other sources. Below shows the screenshots of the project, the full source code can be accessed from my Github Account.

Below are the following steps involved in the development cycle.

  • Importing numpy and pandas used for numeric and data processing.
  • Read in your data.
  • Get rid of null values in various columns of the datasets using "Dropna" function.
  • Remove all outliers. Outliers occur due to miscalculation when collecting the datasets, and can reduce the accuracy of our model.
  • Perform data Normalization/Standardization using the "MinMaxScaler", this would help to keep the values of each dataset at the same base level.
  • Splitting your dataset into training dataset and testing datasets.
  • Performing Data Feature Selection using "SelectKBest" function that selects the number of independent variables best suited for predicting the outcome which is the dependent variable.
  • Choosing one variable best suited for prediction from the selected variables.
  • Using the various algorithms on the variable chosen to predict the outcome and calculate the accuracy score, and time it takes for each algorithms.
