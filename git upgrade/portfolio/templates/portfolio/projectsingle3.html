Data Analysis Project

COVID 19 Patients Data Analysis

Data Analysis of COVID-19 Cases in India, Korea, Italy, and Wuhan using matplotlib, plotly, scattterplot, etc...

Data analysis is a process of inspecting, cleansing, transforming and modeling data with the goal of discovering useful information, informing conclusions and supporting decision-making. In today's business world, data analysis plays a role in making decisions more scientific and helping businesses operate more effectively

Due to the rise in the pandemic called COVID-19 that has plagued most countries in the world i wanted to create a project that shows the progression and analyse the weak spot within different part of a country. My case studies were India, italy, wuhan, and korea. I chose them because as at the time i developed the project those four countries had one of the highest population of COVID-19 cases.

I made use of jupyter notebook to write the code for this project. Jupyter Notebook is an open-source web application that allows you to create and share documents that contain live code, equations, visualizations and narrative text. Uses include: data cleaning and transformation, numerical simulation, statistical modeling, data visualization, machine learning, and much more. I make use of jupyter notebook for my data science projects and visual studio code for my web projects.

I made use of various data visualization algorithms for machine learning like plotly, barchart, scattterplot, matplotlib. Below shows the screenshots for the projects.
